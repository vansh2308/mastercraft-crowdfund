# Easy Bank Landing page

## Features
- The website adapts to device's viewport i.e. fully responsive
- Validates user inputs and data interactivity
- Multistep form and thanks note 
- The user is able to 
  - Hover & Focus over interactive components
  - View the pledge options and select
  - Toggle among various pledge options
  - Customize the pledge amount as per wish 
  - See the updated data - total amount, reduced Offer qty.

## Usage

Navigate following commands in your terminal 

```bash
git clone https://github.com/vansh2308/mastercraft-crowdfund.git
cd ./social-media-links-component
npm i 
npm run dev 
```

Now type http://localhost:5173 in your browser. Bingo!!
